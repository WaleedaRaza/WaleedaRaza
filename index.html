Hand-Gesture Recognition app

Tool created to allow common mac shortcuts to be done via simple hand symbols and motions click here to access repository. Video Demo (click here). Github repository (click here).

Built with python's opencv computer vision library, and google's mediapipe (python lib). this project uses the opencv/mediapipe library to use computer vision to track hand movements live, allowing the script to execute shortcuts like copy, paste, switch desktop & select all. this accessibly is performed with just the movement of the hand, 2 fingers pointed to right or left moves desktops, 3 fingers in the air selects all, index finger up copies, pinky finger up pastes. created this to simplify my user experience on my mac book pro.
